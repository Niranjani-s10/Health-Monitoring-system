# Health Monitoring System

## Project Overview

The Health Monitoring System is a web-based application developed to monitor and manage patient health information efficiently. It allows users to store, manage, and track health-related data through an easy-to-use interface. The system helps in organizing patient details and monitoring health parameters for better healthcare management.

## Features

* User registration and login system
* Store and manage patient information
* Health data monitoring and tracking
* Database integration for secure data storage
* User-friendly interface
* Cloud deployment using AWS

## Technologies Used

* **Frontend:** HTML, CSS
* **Backend:** Python, Flask
* **Database:** MySQL
* **Cloud Platform:** AWS EC2

## Installation Steps

1. Clone the repository:

```bash
git clone <repository-link>
```

2. Move to the project folder:

```bash
cd Health-Monitoring-System
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open in browser:

```bash
http://127.0.0.1:5000
```
